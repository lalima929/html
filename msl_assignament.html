#Write short notes about the selectors, traversal and manipulator in JQuery. 1. jQuery Selectors Selectors are the "factory functions" in jQuery, used to find or select HTML elements from the DOM. They follow the syntax $(selector).action(), where the $ defines jQuery and the selector locates the element. Basic Selectors: Element Selector: Selects all elements with a specific tag name. Example: $("p") selects all

tags. #ID Selector: Uses the # symbol to find a single unique element. Example: $("#myID"). Class Selector: Uses the . symbol to find all elements with a specific class. Example: $(".myClass"). Positional & Attribute Selectors: Position: :first, :last, :even, :odd, and :eq(index) allow filtering based on order. Attribute: $("[href]") selects elements with a specific attribute, while $("[type='text']") filters by value. 2. jQuery Traversal Traversal refers to "moving through" the DOM tree to find elements based on their relationship to other elements (ancestors, descendants, or siblings). Ancestors (Moving Up): .parent(): Returns the direct parent element. .parents(): Returns all ancestor elements up to the root (). .closest(): Finds the first ancestor that matches a specific selector. Descendants (Moving Down): .children(): Returns all direct children of the selected element. .find(): Searches for all descendants (children, grandchildren, etc.) that match a criteria. Siblings (Moving Sideways): .siblings(): Returns all sibling elements of the selector. .next() / .prev(): Returns the immediately following or preceding sibling. 3. jQuery Manipulators Manipulation methods allow you to change the structure, content, or style of the webpage dynamically. Content Manipulation (Getters/Setters): .html(): Gets or sets the inner HTML (includes tags). .text(): Gets or sets only the plain text content. .val(): Gets or sets values from form fields like textboxes. DOM Insertion (Inside/Outside): .append() / .prepend(): Inserts content at the end or beginning inside the element. .after() / .before(): Inserts content outside (before or after) the selected element. Attributes & CSS: .attr(): Used to get or set attribute values (like src or href). .addClass() / .removeClass(): Dynamically toggles CSS classes. Removal: .remove(): Deletes the selected element and its children. .empty(): Removes only the child elements, keeping the selected element itself.
